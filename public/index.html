Mad Lib Project

Last updated:

Addendum: This Mad Lib Project allows users to generate a story by inputting various words (adjective, color, noun, adverb, verb) or by clicking the "Random Word" button to generate a random word for each category. The JavaScript function "generateRandomWord" selects a random word from predefined lists based on the input category. The main challenge faced during development was ensuring the random word generation worked correctly and that the lists of words provided enough variety for interesting story creation.
